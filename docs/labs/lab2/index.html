Лабораторная работа №2

Численные вычисления и анализ данных с использованием NumPy

Бондарева Ирина | 2026

Описание задачи

Цель лабораторной работы — освоить основные возможности библиотек NumPy, Pandas, Matplotlib и Seaborn для выполнения численных вычислений, статистического анализа данных и визуализации результатов.

Основные задачи:

  • Создание и обработка массивов: создание векторов и матриц, изменение формы массивов, транспонирование.
  • Векторные операции: сложение, умножение на скаляр, поэлементное умножение, скалярное произведение.
  • Матричные операции: умножение матриц, вычисление определителя, нахождение обратной матрицы, решение систем линейных уравнений.
  • Статистический анализ: загрузка данных из CSV, вычисление статистических показателей (среднее, медиана, стандартное отклонение, перцентили), нормализация данных.
  • Визуализация: построение гистограммы распределения, тепловой карты корреляции и линейного графика.

Решение задачи

1. Подготовка окружения

Для работы был создан виртуальное окружение и установлены необходимые зависимости:

pip install numpy matplotlib seaborn pandas pytest

В начале файла main.py настроен неинтерактивный бэкенд Matplotlib для корректной работы с сохранением графиков:

import matplotlib
matplotlib.use('Agg')  # Использование Agg backend для сохранения без отображения

2. Реализация функций

Создание массивов

create_vector() — создание вектора от 0 до 9 с помощью np.arange(10).

create_matrix() — генерация случайной матрицы 5x5 с помощью np.random.rand(5,5).

reshape_vector() — изменение формы массива (10,) → (2,5) через np.reshape().

Векторные и матричные операции

Все операции реализованы с использованием векторизации NumPy:

  • vector_add() — поэлементное сложение через оператор +
  • scalar_multiply() — умножение вектора на число через *
  • dot_product() — скалярное произведение через np.dot()
  • matrix_multiply() — умножение матриц через np.matmul() или оператор @
  • matrix_determinant() — определитель через np.linalg.det()
  • matrix_inverse() — обратная матрица через np.linalg.inv()
  • solve_linear_system() — решение СЛАУ через np.linalg.solve()

Статистический анализ

load_dataset() — загрузка данных из CSV с помощью pd.read_csv() и преобразование в NumPy массив.

statistical_analysis() — вычисление статистических показателей.

normalize_data() — Min-Max нормализация по формуле (x - min) / (max - min) с обработкой случая, когда все значения одинаковы.

Визуализация

plot_histogram(): построение гистограммы распределения оценок по математике с использованием plt.hist().

plot_heatmap(): построение тепловой карты корреляции с помощью sns.heatmap().

plot_line(): построение линейного графика зависимости оценок студентов.

Все графики сохраняются в папку plots/ с закрытием фигур через plt.close() для предотвращения утечек памяти.
